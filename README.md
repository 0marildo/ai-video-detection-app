# AI Video Detection

Detect AI-generated videos using pixel-level analysis. Upload a video and get a confidence score indicating whether the content was generated by AI models such as Sora, Veo, Kling, Runway, or Pika.

## Features

- Upload videos (MP4, MOV, AVI, MKV, WebM) up to 100 MB
- AI detection powered by Sightengine's `genai` model
- Confidence score (`0.0` to `1.0`) per analysis
- Async processing with webhook callbacks for longer videos
- User authentication with JWT (access + refresh tokens)
- Personal dashboard with analysis history and statistics
- Rate limiting to prevent abuse
- Docker Compose setup for local development

---

## Architecture

```
Client (Next.js on Vercel)
        |
        v
  FastAPI Backend (Railway)
        |
        +---> Sightengine API (video analysis)
        |
        +---> PostgreSQL (users, analyses, tokens)
```

Videos are sent directly to the Sightengine API for analysis. For longer videos, Sightengine processes asynchronously and delivers results via a webhook callback to the backend.

## Getting Started

### Prerequisites

- Python 3.11+
- Node.js 18+
- PostgreSQL 16+
- A [Sightengine](https://sightengine.com) account (free tier available)

### Environment Setup

Copy the example environment file and fill in your values:

```bash
cp .env.example .env
```

See [`.env.example`](.env.example) for all required and optional variables.

### Backend

```bash
# Install dependencies
pip install -r requirements.txt

# Run database migrations
alembic upgrade head

# Start the server
uvicorn main:app --reload
```

The API will be available at `http://localhost:8000`.

### Frontend

```bash
cd frontend
npm install

# Create local env file
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local

npm run dev
```

The frontend will be available at `http://localhost:3000`.

## API Endpoints

All endpoints are prefixed with `/api/v1`.

### Authentication

| Method | Endpoint         | Description              |
|--------|------------------|--------------------------|
| POST   | `/auth/register` | Create a new account     |
| POST   | `/auth/login`    | Login and receive tokens |
| POST   | `/auth/refresh`  | Refresh access token     |
| POST   | `/auth/logout`   | Revoke refresh token     |

### Analysis

| Method | Endpoint                  | Description                      |
|--------|---------------------------|----------------------------------|
| POST   | `/analyze/upload`         | Upload a video for analysis      |
| GET    | `/analyze/status/{id}`    | Check analysis status and result |
| GET    | `/analyses`               | List user's analysis history     |

### Dashboard

| Method | Endpoint     | Description                          |
|--------|--------------|--------------------------------------|
| GET    | `/dashboard` | Get analysis stats (total, AI, human)|

### Callbacks

| Method | Endpoint                | Description                          |
|--------|-------------------------|--------------------------------------|
| POST   | `/callback/sightengine` | Webhook receiver for async results   |

### Health

| Method | Endpoint   | Description        |
|--------|------------|--------------------|
| GET    | `/health`  | Health check       |

## Docker Setup

Run the full stack with Docker Compose:

```bash
docker compose up --build
```

This starts PostgreSQL, the FastAPI backend, and the Next.js frontend. Make sure your `.env` file is configured before running.

---

## Testing

End-to-end tests use Playwright:

```bash
# Install Playwright browsers
npx playwright install

# Run tests (backend and frontend must be running)
npx playwright test

# Or run the Python test suite
pytest test_ai_detector.py
```

Set `TEST_EMAIL` and `TEST_PASSWORD` environment variables for test credentials.
